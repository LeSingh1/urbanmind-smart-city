Use this stack for the interactive map

|Feature|Tool|
|---|---|
|Real interactive map|**MapLibre GL JS**|
|Draw roads, zones, buildings, markers|**Terra Draw / maplibre-gl-terradraw**|
|Store changed infrastructure|**Supabase**|
|Geo queries like “nearest hospital”|**Supabase PostGIS**|
|Distance/coverage analysis in browser|**Turf.js**|
|Export final city plan|**GeoJSON**|
